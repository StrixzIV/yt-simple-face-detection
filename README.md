# yt-simple-face-detection

## วิธีลง OpenCV

พิมพ์/copy คำสั่งนี้ลงไปใน Terminal/CMD แล้วกด Enter เพื่อรัน

```
  pip install opencv-contrib-python
```

ในกรณีที่จะโหลดจาก conda

```
  conda install opencv
```
